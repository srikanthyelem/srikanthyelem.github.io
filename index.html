Softwares that I have used through out the Real-Time Projects and simulations

PSCAD

Used for high-fidelity EMT simulations, analyzing fast-switching events, harmonics, and the behavior of inverter-based resources like wind and solar.

PSS®E

Primarily used for large-scale transmission planning, power flow, and dynamic stability studies. It helps utilities assess grid-wide impacts and ensure NERC compliance.

ASPEN OneLiner

Specifically for short-circuit calculations and relay coordination, to ensures that protective relays operate correctly during faults to prevent system damage.

Python

Used as a powerful scripting language to automate workflows across these tools.

For example, engineers use the etapPy API to automate reports or the PSS®E Python API to run thousands of contingency scenarios without manual input.

These software packages and programming languages are essential tools for power system analysis, used by engineers to design, simulate, and manage electrical grids.
